## Screen design blueprint

### Dashboard
- KPI cards: streak, phase, target calories, predicted weight.
- CTA: "Start Today’s Workout".
- Micro insight card: "Recovery-adjusted intensity".

### Workout Generator
- Generator prompt panel (goal, time, equipment, soreness, focus).
- Plan preview with exercise cards and set scheme.
- Save as template / send to calendar.

### Workout Session
- Sticky exercise header + current set row.
- One-tap quick log actions (+1 rep, +2.5kg, complete).
- Auto rest timer with skip/extend controls.

### Exercise Library
- Search and faceted filters (muscle, equipment, difficulty, movement).
- Exercise detail tabs: instructions, muscles, media, history.

### Calendar
- Monthly view with workout badges and diet phases.
- Day detail sheet: planned workout, predicted weight, notes.

### Progress
- Weight, strength, volume trend charts.
- PR feed and consistency heatmap.

### Profile & Settings
- Body metrics, goal timeline, training preferences.
- Units, notifications, integrations, export.
