# API Endpoints

## Workout logging
- `POST /workouts/start`
- `POST /workouts/{workout_id}/exercise`
- `POST /workouts/{workout_id}/set`
- `POST /workouts/{workout_id}/finish`

## Routine system
- `POST /routines`
- `POST /routines/{routine_id}/exercise`
- `POST /routines/generate`

## AI assistant
- `POST /assistant/ask`

## Health
- `GET /health`
